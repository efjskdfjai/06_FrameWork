프로필 이미지 사용자가 등록하고, 본인이 등록한 이미지를 마이페이지 확인하는 절차

회원이 선택한 이미지(회원 컴퓨터에 저장된 이미지) 제출 시 --> 서버
서버는 회원이 요청 보낸 이미지 파일을 "서버 컴퓨터의 별도의 경로(C:/uploadFiles/myPage/)"에 저장

회원은 내가 이전에 선택한 이미지가 마이페이지에서 프로필 이미지 부분에 출력될 것을 기대함
클라이언트는 화면에 <img src="경로"> 통해서 서버 컴퓨터에 저장 된 이미지를 노출시킬것을 요청함
서버는 서버 컴퓨터에 저장 된 경로의 이미지를 찾아 클라이언트에게 응답해 줌
클라이언트는 응답받은 경로를 <img src>에 이용하여 화면상에 노출시킴

<img src="/myPage/profile/꽃.jpg">

/myPage/profile/** 모든 요청을 서버 컴퓨터의 C:uploadFiles/myPage/ 폴더로 연결해주는 중간 역할자가 필요
/board/** 모든 요청을 서버 컴퓨터의 C:/uploadFiles/board/ 폴더로 연결해주는 중간 역할자가 필요

스프링에서 이미지를 다루고자 함 이미지는 파일 

파일은 크기가 있음 (서버를 통해 통신한 데이터인 이 파일의 크기를 얼마로 제한할지, 임시 저장경로,,,설정)

1. 파일을 디스크에 쓸 때까지의 임계값 

업로드 되는 파일은 메모리에 저장 (50MB)하다가 초과 시 디스크에 저장

2. HTTP 요청 당 최대 크기 

3. 개별 파일 당 최대 크기

4. 임계 값 초과 시 파일의 임시 저장 경로(서버 컴퓨터 내의 모 경로)
